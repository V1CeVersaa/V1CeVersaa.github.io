# 机器学习

!!! Abstract

    这是我学习周志华教授的《机器学习》一书时做的课程笔记。《机器学习》这本书的工具书属性很浓，但是讲解和推导细腻入微，阅读起来很舒适。
    
    或许这可以作为在 2025 年秋冬学期修读「机器学习」课程的参考资料。

    在目录上的就是学完了的，由于时间原因，很多章节的内容还不是很全，后续随缘完善。
